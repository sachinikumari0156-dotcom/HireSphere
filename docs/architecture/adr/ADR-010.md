# ADR-010 — LocalDB development environment

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Standardize development on (localdb)\MSSQLLocalDB database HireSphereDev.

## Alternatives considered

- Shared Express instance only: less portable.
- Docker SQL: optional, not required for verified path.

## Consequences

Reproducible migrations and UAT on student machines.

## Security impact

Keep connection strings out of git; use user secrets / ignored local files.

## Limitations

Not a production high-availability database.
