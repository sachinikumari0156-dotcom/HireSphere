# ADR-012 — Human-review decision policy

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

AI ranking/match outputs are advisory; Hiring Manager recommendations are advisory; final hire/reject requires authorized Admin/Recruiter final-decision path.

## Alternatives considered

- Fully automated hiring: ethically and academically unacceptable.

## Consequences

UI notices and API separation of recommendation vs final decision.

## Security impact

Reduces unfair automated exclusion risk; audit trail on finals.

## Limitations

Does not replace organizational HR policy outside the system.
