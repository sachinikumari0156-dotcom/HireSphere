# ADR-002 — React frontend

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Use React (Vite) SPA with React Router for four role portals and public pages.

## Alternatives considered

- Blazor WASM: not selected (existing React codebase).
- Next.js SSR: unnecessary for LocalDB coursework demo.

## Consequences

Fast UI iteration; Vitest + Playwright support; client auth state in AuthContext.

## Security impact

Tokens must not leak to XSS; protect routes client-side and enforce server-side.

## Limitations

SEO for public pages is limited without SSR.
