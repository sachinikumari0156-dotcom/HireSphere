# ADR-011 — Direct-main coursework workflow

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

For this completion track, verified Chinthaka commits push directly to main without force-push or history rewrite.

## Alternatives considered

- Long-lived feature branches: used historically (Kalani branch) then merged.
- Force-push: forbidden.

## Consequences

Clear linear verification commits; preserves Kalani authorship on historical commits.

## Security impact

Requires clean gates before push to avoid broken main.

## Limitations

No PR review gate on this track; discipline relies on phase checklists.
