# ADR-004 — SQL Server and EF Core

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Persist domain state in SQL Server (LocalDB for development) via EF Core migrations.

## Alternatives considered

- SQLite-only: insufficient for SQL Server-specific coursework verification.
- Raw ADO: slower iteration.

## Consequences

Migrations and LocalDB verified; integration tests use SQL Server behavior.

## Security impact

Parameterized queries via EF reduce injection risk; still validate inputs.

## Limitations

Cloud managed SQL not verified in this project.
