# ADR-003 — ASP.NET Core backend

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Use ASP.NET Core Web API with controllers, DI, Swagger, and EF Core.

## Alternatives considered

- Node API: would duplicate language skills already in C# coursework path.
- Minimal APIs only: controllers retained for clarity.

## Consequences

Strong typing, mature auth middleware, LocalDB alignment.

## Security impact

Centralized authn/z, validation, and audit hooks.

## Limitations

Requires .NET SDK on developer machines.
