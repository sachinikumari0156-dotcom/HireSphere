# ADR-008 — Notification outbox

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Persist outbound notifications in NotificationOutbox and process asynchronously with retries.

## Alternatives considered

- Synchronous SMTP only: fragile under failures.
- External queue (Service Bus): unnecessary for LocalDB scope.

## Consequences

Idempotent enqueue; delivery status visible; works with mock providers.

## Security impact

Avoids silent loss of audit-relevant communications.

## Limitations

Production scale of outbox not load-tested beyond smoke.
