# ADR-007 — Deterministic AI fallback

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Ship deterministic resume parse/match/rank providers for verified demos; keep external AI optional.

## Alternatives considered

- Mandatory paid LLM: blocked without credentials and cost control.

## Consequences

Repeatable tests and explainable outputs with human-review notices.

## Security impact

Avoids opaque model decisions without review; excludes protected characteristics by design intent.

## Limitations

External AI quality not verified.
