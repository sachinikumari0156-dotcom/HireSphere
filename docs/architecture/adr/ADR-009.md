# ADR-009 — Secure document storage

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Store files via IFileStorageProvider with server-side validation, generated keys, and DB metadata; prefer local/Azurite in coursework.

## Alternatives considered

- Public blob URLs without auth: rejected.
- DB BLOBs only: poorer for large resumes.

## Consequences

Ownership checks on download; invalid uploads rejected.

## Security impact

Path traversal and MIME checks; antivirus NotConfigured unless scanner present.

## Limitations

Azure Blob cloud Not Configured without genuine cloud test.
