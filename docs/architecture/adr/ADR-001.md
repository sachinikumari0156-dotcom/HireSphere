# ADR-001 — Architectural style — modular monolith

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Implement HireSphere as a modular monolith: one ASP.NET Core host with clear controller/service/provider boundaries, plus a separate React SPA.

## Alternatives considered

- Microservices: rejected for coursework scope and operational cost.
- Server-rendered MVC only: rejected to keep SPA role portals.

## Consequences

Simpler deploy and testing; shared DB transactions; must keep module boundaries disciplined.

## Security impact

Single trust boundary for API; JWT still required at edge.

## Limitations

Horizontal scaling of independent services is not available without extraction.
