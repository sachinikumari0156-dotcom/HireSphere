# ADR-005 — JWT and RBAC

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Authenticate with JWT Bearer tokens; authorize via role claims and resource ownership checks.

## Alternatives considered

- Cookie sessions only: less convenient for SPA + API split.
- Refresh-token rotation: deferred optional bonus.

## Consequences

Stateless API auth; four roles enforced; disabled users rejected.

## Security impact

Short-lived access tokens; no password hashes in DTOs; IDOR tests required.

## Limitations

Refresh tokens not implemented; logout is client discard + audit.
