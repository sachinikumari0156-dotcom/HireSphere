# ADR-006 — Modular provider abstractions

**Status:** Accepted  
**Date:** 2026-07-21  
**Author:** Chinthaka Jayaweera  

## Context

HireSphere is a coursework recruitment platform requiring four role portals, secure data handling, and optional external integrations.

## Decision

Abstract email, SMS, calendar, storage, antivirus, and AI behind interfaces with NotConfigured and development adapters.

## Alternatives considered

- Hard-wire cloud SDKs: blocks local demos without credentials.

## Consequences

Truthful provider status endpoints; safe coursework demos.

## Security impact

Prevents accidental calls to unpaid external APIs when NotConfigured.

## Limitations

External providers remain unverified until configured.
