# Design tokens

CSS custom properties under `:root` in `Frontend/src/styles/tokens.css`.

| Category | Examples |
|----------|----------|
| Brand | `--hs-brand-500` … `--hs-brand-800` |
| Neutrals | `--hs-neutral-0` … `--hs-neutral-900` |
| Semantic | `--hs-success-*`, `--hs-warning-*`, `--hs-error-*`, `--hs-info-*` |
| Text/surfaces | `--hs-text`, `--hs-bg`, `--hs-border`, `--hs-focus` |
| Spacing | `--hs-space-1` … `--hs-space-12` |
| Radius/shadow | `--hs-radius-*`, `--hs-shadow-*` |
| Typography | `--hs-font-sans`, `--hs-font-display`, `--hs-text-*`, weights |
| Layout | `--hs-max-content`, `--hs-nav-height`, `--hs-sidebar-width` |
| Motion | `--hs-duration-*`, `--hs-ease` |
| Z-index | `--hs-z-sticky`, `--hs-z-modal`, `--hs-z-toast` |

Contrast: primary text (`--hs-neutral-800`) on `--hs-bg` / elevated white meets AA for normal text. Focus ring uses teal with offset outline.
